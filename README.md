# WebServerWithGoogleAuth

https://console.developers.google.com/

## Sample

Create `THIS_REPOSITORY_DIR/config.json` .

```json
{
	"googleauth":
	{
		"client_id": "YOUR_CLIENT_ID",
		"client_secret": "YOUR_CLIENT_SECRET",
		"baseurl": "http://localhost:18080",
		"users": [ "YOUR_ADDRESS" ]
	},
	"port": 18080,
	"debug": true
}
```
